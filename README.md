## 🚀 三步部署

### 步骤1：确认服务器类型
- Windows Server + IIS
- Linux + Nginx
- Linux + Apache

### 步骤2：阅读部署文档
打开 `部署指南.md`，找到对应服务器的部署方法

### 步骤3：联系IT部署
将 `IT部门沟通文档.md` 发给IT部门，协助完成部署

## 🔧 系统特点

- **零依赖：** 纯静态文件，无需数据库
- **跨平台：** 支持所有主流Web服务器
- **易维护：** 更新问答只需替换JSON文件
- **安全：** 内网部署，数据隔离
- **响应式：** 支持电脑和手机访问

---

## 📝 如何更新问答内容

1. 在管理后台（admin.html）编辑问答
2. 点击"导出数据"按钮
3. 将导出的文件改名为 `qa-data.json`
4. 替换服务器上的 `qa-data.json` 文件
5. 用户刷新页面即可看到更新
